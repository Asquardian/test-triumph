Сохранить - Сохранить раб. область в local storage

Сбросить - Удалить данные из local storage

Создать - Создать новые фигуры, количество зависит от введенного значения в текстовом поле

В текстовом поле можно выбрать от 5 до 20 фигур

С помощью колесика мыши можно изменять масштаб

С помощью ЛКМ можно перемещать раб область по контейнеру

Если зажать ЛКМ и покрутить колесиком мыши, то можно прокрутить контейнер
